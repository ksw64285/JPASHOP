## ⏰ 프로젝트 회고

### 👍 좋았던 점

- 회원 상품 주문에 대해 도메인과 테이블을 설계해보며 공부하였습니다
- 회원 상품 주문 도메인의 비즈니스 로직을 개발하며 공부하였습니다
- 타임리프를 사용하여 웹 화경에서 동작하도록 공부하였습니다

### 🤦‍♂️ 아쉬웠던 점
- 상속관계 매핑 전략 (JOINED, SINGLE_TABLE, TABLE_PER_CLASS)을 잘 이해하지 못한 것이 아쉬움에 남습니다.

## ⚙️ Environment
- Spring Boot 2.7.12
- Java 11
- H2 Database 1.4.199
